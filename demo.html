安装完整版本

1

安装 Node.js

访问 nodejs.org 下载并安装 Node.js LTS 版本

2

安装依赖

在项目目录执行 npm install 安装所需依赖包

3

启动应用

执行 npm start 或双击 start.bat 启动应用

4

访问应用

打开浏览器访问 http://localhost:3000

快速安装命令：

# 安装依赖
npm install

# 启动应用
npm start

# 或者使用启动脚本
# Windows: 双击 start.bat
# Linux/macOS: ./start.sh
